o que é?

É um framework para desenvolvimento front-end, que utiliza a abordagem Mobile First, possibilitando assim, otimizar o processo de criação de sites responsivos.

Criado em 2011 por dois integrantes do Twitter, para facilitar seus trabalhos. Em 2012 foi disponibilizado o código fonte do Bootstrap, tornando-a assim uma ferramenta opensource.

Por ser compatível com HTML5, CSS3 e JQuery, e funcionar nos principais navegadores atuais, se popularizou rapidamente no meio web.

Por que usar?

Por que é uma boa ideia usar o bootstrap hoje em dia?
Por que tantos desenvolvedores se renderam ao uso da ferramenta?

HTML5 CSS3 JQuery

Bootstrap utiliza CSS3 e plugins Javascript (JQuery), permitindo criação de sites HTML5. Portanto, são reconhecidas pelos principais navegadores da atualidade (Chrome, Firefox, Safari, IE!).

'

Acessível

Por utilizar HTML5, é acessível por diferentes dispositivos. Desde as SmarTV's - que ainda estão se popularizando -, até os milhares de modelos diferentes de Smartphones.

Desenvolvimento Ágil

Esse é um dos principais objetivos do Bootstrap. Agilizar o desenvolvimento de páginas web, disponibilizando componentes estilizados e prontos para usar.

SEO (Google)

Em buscas feitas no Google a partir de dispositivos mobile, serão mostrados resultados apenas de sites que são responsivos. Se for um site antigo, seu site aparecerá apenas nos computadores.

Por que não usar?

Por que nem todo mundo gosta do Bootstrap?
Pura implicância, ou existe um motivo obscuro?

Sites Padronizados

Por disponibilizar componentes prontos, se não forem feitas alterações no estilo do site, ele ficará muito parecido com os outros sites.

Código Poluído

Por vezes são necessários muitos elementos HTML para fazer um simples bloco de texto ou imagem, deixando a estrutura do site poluída.

Conhecimento Prévio

Apesar de ter o objetivo de agilizar o desenvolvimento, pode ser complicado entender o framework se não possuir conhecimento prévio das tecnologias usadas.

Como usar?


Como começar a usar o Bootstrap?

Fazendo o download do Bootstrap e colocando a mão na massa.
E mantendo a documentação do framework sempre às mãos!

Ou então usando uma ferramenta que facilite ainda mais, como Mobirise, LayoutIt! e Bootply. Com elas é possível montar sites apenas por meio de Drag & Drop.

Conclusão


O Bootstrap pode ser uma ótima opção para agilizar o desenvolvimento de projetos web, mas também pode ter efeitos indesejados, como deixar seu site muito parecido com outros sites. Portanto, o melhor é analisar se os benefícios alcançados pelo seu uso supera as desvantagens, antes de começar o desenvolvimento do projeto.
